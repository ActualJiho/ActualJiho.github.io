What is difference between Margin, Border, and Padding?

Margin, border, and padding are all CSS properties used for defining the spacing and appearance of elements on a webpage.


Margin

Margin is the space outside of element's border. It's the distance between the element and the adjacent elements. Margin can be set for all four sides separately or at once.

margin-top:

margin-bottom:

margin-left:

margin-right:

or,

margin:   ;


You can also set the margin property using either one, two, three or four values.


margin: 10px;   sets all four sides of margin to 10px.

margin: 10px 20px;   first value sets top and bottom margin to 10px and second value sets left and right margin to 20px.

margin: 10px 20px 30px;   first value sets top margin to 10px, second value sets left and right margin to 20px and third value sets bottom margin to 30px.

margin: 10px 20px 30px 40px;   Using four values sets the top, right, bottom, and left margins, respectively.


You can also set a negative margin which will cause element to overlap to the next element.

This is similar to the whitespace you see when you write paragraphs in a Microsoft Word document.


Border

Border is a line of box around an element's border and it seperates margin and padding. Think of it as a frame for a picture.


With border, you need to define at least one css rule to make it visible otherwise border will not display.


border: 10px;   sets all four sides of border to 10px.

border: 10px solid;   sets the border to 10px with solid line.

border: 10px solid #000;   sets the border to 10px with black solid line.


Do note that orders of the values do not matter.


Padding

Padding is the space inside of element's border. Padding can be set for all four sides separately or at once just like margin. The padding is inside the border, but outside the actual content.



border-top:

border-bottom:

border-left:

border-right:

or,

border:   ;


You can also set the padding property using either one, two, three or four values just like margin!.

However, unlike margin you can't set the padding with negative value as it will either be ignored to treat it as 0 value.
