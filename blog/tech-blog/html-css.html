What is difference between Marin, Border, and Padding?

Margin

Margin is the space outside of element's border. It's the distance between the element and the adjacent elements. Margin can be set for all four sides separately or at once. You can also set a negative margin which will cause element to overlap to the next element. It is same as the whitespace you see when you write paragraphs in Microsoft Word document.

Border

Border is a line of box around an element's border and it seperates margin and padding. Think of it as a frame for a picture.

Padding

Padding is the space inside of element's border. Padding can be set for all four sides separately or at once just like margin. The padding is inside the border, but outside the actual content.
