HTML vs CSS vs Javascript

HTML is like the builder of a house. They are the one laying the foundation and building the sturction of the house. Just like in web development HTML is responsible for building the structure of the page.

Whereas CSS is like the designer. They are responsible for adding style and asthetics to the house that HTML/Builder built. Such as painting the house and selecting type of flooring. In web development, CSS is responsible for adding styles like fonts,colors and positioning.

Finally, Javascript can be compared to plumber or electrician. They make sure all the plumbing and wiring is connected together and working properly. Such as pipes, gas, drains, lights and so on. In web development Javascript makes the page dynamic.


Control flow and Loops

Control flow is like a schedule or recipe that tells a computer what to do step by step.

For example, we can use the process of going to work each day:


  1. Wake up and get ready
  2. Get in the car
  3. Drive to work
  4. Do work
  5. Drive back home
  6. Get cleaned and go to sleep

We can also use the same example to explain loops. A loop is a programming construct that tells the computer to repeat a specific set of instructions multiple times.

Using the same example, a loop would be like telling someone to repeat the above process of going to work until they run out of gas.

In javascript, we can set it's starting point, end point(so it doesn't loop indefinitely) and what actions to take during the loops.

Document Object Model(DOM)

DOM is a map for web developers. It will show all the different parts of the web page. It will help us developers to change or move stuff on the web using a tool called Javascript.

I can give you an example. If a web page has a button and when clicked change its colour. For web developer we can use the DOM to tell the program to listen to that click and to change the color of the button.

Obiously this is one of many examples out there. Javascript has many different functions we can utilise to make interactive things on a web page.


Arrays and Objects

Arrays are like box full of drill bits. As they are numbered you can easily find the right size you are looking for.

Unlike the example I've given you, javascript starts counting the items from 0 instead of 1. so if we have 5 values in an array. it would count them as 0,1,2,3,4 instead of 1,2,3,4,5

Now let's talk about objects. Objects can be compared to a tool drawer with name labels on each drawers so you can open the right drawer.For example, you might have a drawer labeled "screwdrivers" and another labeled "allen keys". Each drawer would contain the tools that match its label,

It's the same in javascript where an object contains a properties that describe its value.

In javascript, you have the ability to modify objects by adding, changing, or removing their properties. which are like the different tools in a tool drawer that can be swapped out or replaced.


Functions

Functions are sets of instructions for Javascript to follow. Following the other two analogy I've given you above for arrays and objects, javascript can be compared to a power drill.

Power drill is a tool that is used with different drill bits to perform many different tasks. Just like in Javascript we use function to perform different tasks and it depands on what arguments are passed on to it.

Function can be reused later on the time when needed, like how power drill can be stored and used many times.
